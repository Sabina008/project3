Pylint – What is it?

Pylint is a static code analysis tool used to detect mistakes in Python code and aids programmers in applying good coding style. This tool assists them in resolving complex code with little manual effort.

This is one of the tools that is used for test-driven development (TDD).

Pylint uses a coding style known as PEP8.

Installing Pylint

Installing Pylint on either system: Windows 10, Mac OS, Linux, use the following command: 

john01 Desktop$ pip install pylint

When running Pylint without any arguments, this will get the help dialogue and provide more details of the issues.

Do this now, i.e.: 

john01 Desktop$ pylint
